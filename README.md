Overview
--------

Simple Vim configuration backup.

Setup
----

```bash
$ git clone https://github.com/srobbibaro/.vim.git
$ ln -s .vim/vimrc ~/.vimrc
$ mkdir ~/tmp
$ git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
$ vim +PluginInstall +qall
```
